[设计模式](https://github.com/ChinesePowerful/gof23)
=======

# 简介

- 设计模式是解决问题的方案，学习现有的设计模式可以做到经验复用
- 设计模式是多数人知晓的，可以降低沟通的成本，不需要了解底层细节

# 创建形
